パーティションの一覧を表示する sfdisk -l / CentOS

(CentOS5.7で確認)

　パーティションの一覧を表示するにはsfdiskコマンドで-lオプションを使用します。 

sfdisk -l

　出力結果は以下のようになります。 

ディスク /dev/hda: シリンダ数 264220、ヘッド数 16、63 セクタ/トラック
Warning: The partition table looks like it was made
  for C/H/S=*/255/63 (instead of 264220/16/63).
For this listing I'll assume that geometry.
ユニット = 8225280 バイトのシリンダ、1024 バイトのブロック、0 から数えます

   Device Boot Start     End   #cyls    #blocks   Id  System
/dev/hda1   *      0+     12      13-    104391   83  Linux
/dev/hda2         13     273     261    2096482+  82  Linux swap / Solaris
/dev/hda3        274   16577   16304  130961880   8e  Linux LVM
/dev/hda4          0       -       0          0    0  空

　「/dev/hda」というデバイス(ハードディスク)に設けられたパーティションの一覧が表示されています。デバイスが複数ある場合は、すべてのデバイスに対してパーティションの一覧が表示されます。

　デバイスを指定して表示することもできます。 

sfdisk -l /dev/hda
