yum - パッケージをインストールする | CentOS

　yumはCentOSで、パッケージをインストールするためのコマンドです。

インストール・アップデート・アンインストール

<h3> インストール</h3>
yum install パッケージ

<h3> アップデート</h3>
yum update パッケージ

<h3> アンインストール</h3>
yum remove パッケージ
yオプションで質問に自動的にイエスと答えることができます。 
yum -y install パッケージ
