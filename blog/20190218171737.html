インデックスに加えた変更を元に戻す

　コミットを行うためにaddコマンドを使用してインデックスを更新したけれど、インデックスに加えた変更を元に戻したい場合があります。そのような場合はresetコマンドを使用します。 

git reset

　これでインデックスに加えた変更が元に戻ります。diffコマンドで確認して見ると、インデックスの状態が元に戻って、作業ツリーとの差分が表示されるのがわかると思います。 

git diff
