コミットされるファイルを確認する git status / Git

　インデックスに追加されてコミットされようとしているファイルを確認するにはgit statusコマンドを使用します。 

git status

　余計なファイルがコミットされようとしていないかを確認できます。git addコマンドでインデックスにコミットするファイルを追加した後に、本当に正しいファイルをコミットするかを確認するのに利用しましょう。間違ってコミットすることを事前に防ぐことができます。
