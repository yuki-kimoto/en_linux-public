Gitのリポジトリの説明(Description)を記述する / Git

　gitweb.cgiやGitweb Liteを使っていると、プロジェクト一覧のページの説明の部分が「Unnamed repository; edit this file to name it for gitweb」となっています。これを記述するには.gitリポジトリの下にdescriptionというファイルを配置して記述します。 

.git/description
