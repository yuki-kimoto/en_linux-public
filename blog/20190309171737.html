コミット数を取得する方法 / Gitメモ

　あるブランチのコミット数を取得するには次のようにする。 

git shortlog -s ブランチ名

　ブランチ名を省略した場合は、現在のブランチになる。

　自分がgitリポジトリのディレクトにいない場合は--git-dirオプションというのでディレクトリを指定できる。 

git --git-dir=ディレクトリ名 shortlog -s ブランチ名

　次のような出力が得られる。 

    25  Yuki Kimoto
     1  root

　コミットした回数とコミットしたユーザーの名前がでてくる。コミット数の合計を出したい場合は、左の値を足し算すればよい。
