PHPみたいにファイルを作ったら、実行されるフレームワークはあるかな -> 作りました。

PHPのファイルを置いたらそのまま実行されるという機能はとても便利だね。PHPの一番の魅力だ。

Perlのフレームワークにも、そういう機能があったらいいな。ファイルがたくさんあると「いちいちファイルごとにURLを定義するなんて、めっちゃめんどうだよー」と思うからね。

これを実現するためのMojoliciousのプラグインを作ったよ。

ルートを自動的に生成するためのプラグイン

　Mojolicious::Plugin::AutoRouteはルートを自動的に生成してくれるプラグインです。テンプレートを置くだけで、Webアプリケーションを作成することができます。

MojoliciousにPHPの機能を組み組むプラグインだよ。

Mojolicious::Plugin::AutoRouteの簡単な使い方

use Mojolicious::Lite;

plugin 'AutoRoute';

これをコードに書いてから。templates/autoというディレクトリにファイルを置くだけ。 

テンプレート                         URL
templates/auto/index.html.ep        # /
              /foo.html.ep          # /foo
              /foo/bar.html.ep      # /foo/bar
              /foo/bar/baz.html.ep  # /foo/bar/baz

Webサイトを作るときは、ちょっとだけ動的で、ほとんど静的というページが大量にできるね。こういうときは、このプラグインを使うと、たくさん早く、ページが作れて、便利だよ。
