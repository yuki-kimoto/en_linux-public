crondでプログラムを自動実行する / CentOS

cronの設定 / CentOS

(CentOS5.7で確認)

　cronとは、プログラムを時間起動で自動実行するためのアプリケーションです。バックグラウンドでデーモンとして起動し、時間がくると設定ファイルに登録されたプログラムを実行します。

##サービス起動スクリプト

　サービス起動スクリプトを見ればいろいろな情報がわかります。サービス名はcrondです。

```

# サービス起動スクリプトを見る。

view /etc/init.d/crond

```

##サービスの自動起動の設定

　自動起動の設定を確認する。

```

# 自動起動の確認

chkconfig --list crond

# 自動起動をonにする

chkconfig --level crond on

# サービスそのものとして登録されていない場合は追加する。

chkconfig --add crond

```

##サービスの開始・停止・確認

```

# サービスが開始しているかどうかの確認

service crond status

# 開始していない場合は開始

service crond start

# 停止する場合は

service crond stop

```

##crondのログファイル

```

view /var/log/cron

```

##crondの設定ファイル

```

view /etc/crontab

```
