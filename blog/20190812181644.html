ファイル(blob)のサイズを取得する / Git

　gitで管理されているあるファイルのサイズを取得するにはどうすればよいでしょう。gitではファイルはblobという名前で呼ばれます。blobのサイズを取得するにはgit cat-fileコマンドで-sオプションを利用します。

```

git cat-file -s master:README

```

　注意しなればいけないことは、ファイル名そのものではなくてオブジェクト名を指定しないといけません。オブジェクト名はリビジョン:ファイル名で指定できます。
