制御文と繰り返し文

　C言語の制御文と繰り返し文を覚えましょう。

##条件分岐 if ～ else if ～ else

　if ～ else if ～ else文です。

```c:

int num = 1;

if (num == 1) {

// ...

}

else if (num == 2) {

// ...

}

else {

// ...

}

```

##条件分岐 switch文

　C言語にはswitch文があります。複数分岐で整数を判定するときはswitch文を使えます。分岐の数が多いときはswitch文のほうがパフォーマンスがよいことが多いようです。breakを書かないと、breakより下の文が実行されてしまうので注意しましょう。

```c:

int num = 3;

switch(num) {

case 1: // ... break; case 2: // ... break; case 3: // ... break; default : // ...

}

```

##繰り返し while

　繰り返しをするときはwhile文を使うことができます。Perlと同じです。

```c:

int num = 0;

while (num < 4) {

// ... num++;

}

```

##繰り返し for

　繰り返しをするときはfor文を使うことができます。Perlと同じです。

```c:

for (int i = 0; i < 5; i++) {

// ...

}

```

##繰り返しからの脱出

　break文を使います。Perlのlastに該当します。

```c:

break;

```

##次のループに進む

　continue文を使います。Perlのnextに該当します。

```c:

continue;

```

##サンプルコード

　条件分岐と繰り返しのサンプルです。

```c:

#include

int main () {

int num = 2; if (num == 1) { printf("num is 1\n"); } else if (num == 2) { printf("num is 2\n"); } else { printf("num is other\n"); } switch(num) { case 1: printf("num is 1\n"); break; case 2: printf("num is 2\n"); break; case 3: printf("num is 3\n"); break; default : printf("num is other\n"); } int k = 0; while (k < 4) { printf("k is %d\n", k); k++; } for (int i = 0; i < 5; i++) { printf("i is %d\n", i); } return 0;

}

```
