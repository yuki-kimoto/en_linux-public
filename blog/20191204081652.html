mvコマンド - ファイル名を変更する

mvコマンドを使用すると、ファイル名を変更することができます。元のファイル名が、新しいファイル名に変更されます。mvの名前はmoveに由来しています。 

mv 元のファイル名 新しいファイル名

mvコマンドでファイル名変更を行うサンプルです。foo.txtというファイル名が、bar.txtに変更されます。 

mv foo.txt bar.txt

ディレクトリ名を変更する

mvコマンドは、ディレクトリ名の変更にも利用することができます。Linuxでは、ファイル名とディレクトリ名は、同じ扱いです。ディレクトリというのは、Windowsでいうフォルダのことです。 

mv 元のディレクトリ名 新しいディレクトリ名

ディレクトリ名の変更行うサンプルです。dir1というディレクトリ名がdir2というディレクトリ名に変更されます。 

mv dir1 dir2

ファイルの場所を移動する

mvコマンドは、ファイルの場所を移動するのにも利用することができます。

ホームディレクトリのlaboというディレクトリの中にあるfoo.txtを、ホームディレクトリのtestというディレクトリに移動してみましょう。「~」はホームディレクトリのパスを表します。 

mv ~/labo/foo.txt ~/test/foo.txt

上記のように、ファイルを移動するときにもmvコマンドは使用します。

ファイルを移動したいときは、末尾のファイル名だけを省略することもできます。末尾にスラッシュが付いていることに注意してください。 

# 上の操作のシンタックシュガー
mv ~/labo/foo.txt ~/test/

ディレクトリの場所を移動したいときも同じように行うことができます。 

mv ~/labo/dir1 ~/test/dir1

または 

# 上の操作のシンタックシュガー
mv ~/labo/dir1 ~/test/

ヘルプを表示する --helpオプション

ヘルプを表示するには「--help」オプションを使用します。 

mv --help

ヘルプが表示されます。 

Usage: mv [OPTION]... [-T] SOURCE DEST
  or:  mv [OPTION]... SOURCE... DIRECTORY
  or:  mv [OPTION]... -t DIRECTORY SOURCE...
Rename SOURCE to DEST, or move SOURCE(s) to DIRECTORY.

Mandatory arguments to long options are mandatory for short options too.
      --backup[=CONTROL]       make a backup of each existing destination file
  -b                           like --backup but does not accept an argument
  -f, --force                  do not prompt before overwriting
  -i, --interactive            prompt before overwrite
      --strip-trailing-slashes remove any trailing slashes from each SOURCE
                                 argument
  -S, --suffix=SUFFIX          override the usual backup suffix
  -t, --target-directory=DIRECTORY  move all SOURCE arguments into DIRECTORY
  -T, --no-target-directory    treat DEST as a normal file
  -u, --update                 move only when the SOURCE file is newer
                                 than the destination file or when the
                                 destination file is missing
  -v, --verbose                explain what is being done
      --help     display this help and exit
      --version  output version information and exit
