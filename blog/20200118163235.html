CentOS 7にcronをインストールする

CentOS 7にcronをインストールしましょう。CentOS 7では、デフォルトでanacronと呼ばれる、別種類のcronがインストールされているので、これを削除することも必要です。 

# 通常のcronのインストール
sudo yum -y install cronie-noanacron

# anacronの削除
sudo yum -y remove cronie-anacron

インストールした後は、crondを再起動しましょう。 

sudo systemctl restart crond

cronのログは以下です。 

sudo tail /var/log/cron
